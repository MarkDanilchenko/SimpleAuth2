## App Description &#128209;

This API based on NodeJS (ExpressJS) and provides endpoints for signIn/signUp with JWT, editing and viewing users profile.

## Launch instructions &#128190;

- ExpressJS;
- PostgreSQL;
- Sequelize;
- Swagger;
- Docker;
